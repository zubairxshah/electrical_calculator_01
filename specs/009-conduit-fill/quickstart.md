# Quickstart: Conduit/Raceway Fill Calculator

**Feature**: 009-conduit-fill | **Branch**: `009-conduit-fill`

## Prerequisites

- Node.js 18+
- pnpm (project package manager)

## Setup

```bash
git checkout 009-conduit-fill
pnpm install
pnpm dev
```

Navigate to `http://localhost:3000/conduit-fill`

## Development Order

Follow the task order in `tasks.md` (generated by `/sp.tasks`). High-level sequence:

1. **Types** (`types/conduit-fill.ts`) — define all interfaces first
2. **NEC Data** (`lib/calculations/conduit-fill/conduitFillData.ts`) — enter NEC table data with tests
3. **Calculator** (`lib/calculations/conduit-fill/conduitFillCalculator.ts`) — core fill logic with tests
4. **Validation** (`lib/validation/conduitFillValidation.ts`) — Zod schemas
5. **Store** (`stores/useConduitFillStore.ts`) — Zustand state
6. **Components** (`components/conduit-fill/`) — UI components
7. **Page** (`app/conduit-fill/`) — page + tool wrapper
8. **Navigation** — add to TopNavigation, Sidebar, homepage
9. **PDF** (`lib/pdfGenerator.conduitFill.ts`) — export feature

## Running Tests

```bash
pnpm test                                          # all tests
pnpm test __tests__/unit/calculations/conduit-fill  # conduit fill tests only
```
